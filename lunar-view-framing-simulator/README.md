# Lunar View Framing Simulator

이 시뮬레이터는 Living Aegis Origin의 공식 게임 화면이 아니라, 달 표면 1인칭 화면 구도를 검증하기 위한 제작 보조 도구입니다.

## 목적

`lunar-view-framing-simulator`는 달 표면에서 지구를 바라보는 1인칭 화면 구도를 빠르게 비교하기 위한 2D Canvas 참고 도구입니다.

검증 초점은 실제 천문 좌표 재현이 아니라, 화면 안에서 지구와 달 표면이 어느 정도 비중으로 보일 때 기본 화면 구도가 답답하거나 비어 보이지 않는지 확인하는 데 있습니다.

## 조정 가능한 값

- 지구 확대율
- 지구 확대율 1x / 3x / 5x / 7x / 10x preset
- 지구의 화면상 세로 위치
- 달 표면이 화면 하단에서 차지하는 비중
- 화면 중앙선, 달 표면 하단 1/4, 달 표면 하단 1/3 guideline 표시 여부

## 사용 방법

브라우저에서 다음 파일을 엽니다.

```text
lunar-view-framing-simulator/index.html
```

슬라이더로 지구 확대율, 지구 위치, 달 표면 비중을 조정합니다. 1x / 3x / 5x / 7x / 10x 버튼은 지구 확대율을 빠르게 바꾸기 위한 preset입니다.

`Guidelines`를 켜면 화면 중앙선과 달 표면 1/4, 1/3 기준선을 함께 확인할 수 있습니다. 현재 설정값은 조정 패널 아래에 표시됩니다.

## 1차 기본 구도 후보

현재 simulator 기준 1차 기본 구도 후보는 다음과 같습니다.

- Earth Scale: 5x 또는 7x
- Earth Vertical Position: 30%
- Lunar Surface Area: 30%

이 값들은 Living Aegis Origin 게임 본체의 공식 확정값이 아니라, simulator 기준의 1차 기본 구도 후보입니다. 이후 prototype에서 실제 조준, 미사일 식별, HUD 배치와 함께 다시 검증해야 합니다.

15x 확대는 기본 구도값이 아니라 향후 줌 연출 또는 관측 모드 후보로 분리합니다. 15x 확대는 현재 simulator 구현 대상이 아닙니다.

## 현재 한계

- 실제 천문 좌표를 계산하지 않습니다.
- 3D 시점을 구현하지 않습니다.
- 광속 지연을 다루지 않습니다.
- 전투/HUD/조준 시스템을 구현하지 않습니다.
- 공식 게임 화면의 최종 기준값을 확정하지 않습니다.

## 상태

1차 구현 상태입니다. Living Aegis Origin의 기본 화면 구도 판단을 돕기 위한 참고용 simulator로 유지합니다.
